Filter

Return only the objects matching certain criteria.

const filteredInventors = inventors.filter(inventor => (inventor.year >= 1500 && inventor.year < 1600));

Map

Transform an array into a new array.

const firstLastNames = inventors.map(inventor => `${inventor.first} ${inventor.last}`);

Sort

Re-order elements based on certain criteria.

const sortedInventors = inventors.sort((a, b) => (a.year > b.year) ? 1 : -1);

      const nextInventor = b.passed - b.year;
      return nextInventor > prevInventor ? 1 : -1;

const sortedPeople = people.sort((a, b) => {
